## What's coming

The Workflows API will let you:

- **Create workflows** — Define multi-step business processes with assignees, SLAs, and rules
- **Run workflows** — Trigger a new live workflow instance from your system
- **Track progress** — Query the current step, pending approvals, and overdue tasks
- **Manage definitions** — Update, archive, and version your workflow templates

## Stay updated

To be notified when the Workflows API launches, reach out to [support@doubletick.io](mailto:support@doubletick.io) or watch the [changelog](https://doubletick.io/changelog).

In the meantime, you can manage workflows directly from the [DoubleTick dashboard](https://app.doubletick.io).
