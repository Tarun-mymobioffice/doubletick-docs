## Before you begin

You need a DoubleTick account and a secret key. Get yours from your [dashboard settings](https://app.doubletick.io/settings/api).

---

## Step 1 — Get your secret key

1. Log in to [doubletick.mobi](https://doubletick.mobi)
2. Go to **Settings → API Keys**
3. Copy your **Secret key**

<Warning>Store your secret key securely. Never expose it in client-side code or commit it to version control.</Warning>

---

## Step 2 — Create your first task

Pass your secret key in the `secret-key` header:

<CodeGroup>

```bash cURL
curl -X POST 'https://prod-api.task-management.mobioffice.io/api/task/InsertTaskMain' \
  -H 'secret-key: <your-secret-key>' \
  -H 'Content-Type: application/json' \
  --data-raw '{
    "IsActive": true,
    "TaskMain": {
      "IsActive": true,
      "TaskIds": [],
      "Title": "My First Task",
      "DescriptionHtml": "<p>Created via API</p>",
      "DescriptionText": "Created via API",
      "AssignedToID": "your.email@company.com",
      "StartDate": "2026-03-28T12:30",
      "EndDate": "",
      "StatusID": 20,
      "FlowID": 2,
      "PriorityID": 4,
      "ParentTaskID": "",
      "ProjectID": 12,
      "AssetID": 394,
      "ReferenceID": "",
      "CustomerID": 0,
      "TaskDetailsConfigurationID": 5,
      "WorkspaceID": 5
    },
    "MentionIds": [],
    "CollaboratorsEmailIds": [],
    "CarbonCopyEmailIds": [],
    "LabelIds": [],
    "TagIds": [],
    "AttachedFiles": [],
    "WorkBatchIDs": [],
    "IsTaskRulesApplied": false,
    "CheckListItems": []
  }'
```

```python Python
import requests

response = requests.post(
    "https://prod-api.task-management.mobioffice.io/api/task/InsertTaskMain",
    headers={
        "secret-key": "<your-secret-key>",
        "Content-Type": "application/json"
    },
    json={
        "IsActive": True,
        "TaskMain": {
            "IsActive": True,
            "TaskIds": [],
            "Title": "My First Task",
            "DescriptionText": "Created via API",
            "AssignedToID": "your.email@company.com",
            "StatusID": 20,
            "WorkspaceID": 5
        },
        "MentionIds": [],
        "CollaboratorsEmailIds": [],
        "LabelIds": [],
        "TagIds": [],
        "AttachedFiles": [],
        "WorkBatchIDs": [],
        "IsTaskRulesApplied": False,
        "CheckListItems": []
    }
)
print(response.json())
```

```javascript Node.js

});
const data = await response.json();
console.log(data);
```

</CodeGroup>

A successful response looks like:

```json
{
  "IsSuccess": true,
  "Message": "Task created successfully",
  "TaskID": 109006
}
```

## Step 3 — Add a comment

Use the `TaskID` returned above to post a comment:

```bash
curl -X POST 'https://prod-api.task-management.mobioffice.io/api/task/InsertTaskComment' \
  -H 'secret-key: <your-secret-key>' \
  -H 'Content-Type: application/json' \
  --data-raw '{
    "IsActive": true,
    "TaskComment": {
      "IsActive": true,
      "TaskID": 109006,
      "CommentHtml": "<p>Task is under review.</p>",
      "CommentText": "Task is under review.",
      "MentionIds": [],
      "ExternalFrom": "",
      "IsIncoming": false,
      "IsPrivate": false,
      "IsSentAsWhatsapp": false
    }
  }'
```
