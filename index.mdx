## Welcome to DoubleTick API

DoubleTick turns your repeatable business processes into tracked tasks and workflows — so approvals move faster, follow-ups happen automatically, and nothing slips through the cracks.

Use the DoubleTick API to create and update tasks, post comments, and integrate your operations directly into your own systems.

## Base URL

All API requests are made to:

```
https://prod-api.task-management.mobioffice.io
```

<Note>
  Every endpoint requires an `Authorization` header. You can use a **Bearer token** or a **secret key**. See [Authentication](/api-reference/introduction) for details.
</Note>
